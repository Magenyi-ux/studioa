# **App Name**: ZediLearn

## Core Features:

- Past Questions Library: Access a comprehensive library of WAEC & JAMB past questions categorized by topic and year.
- Offline AI Assistant: An AI-powered study tool that works offline and syncs when online.
- Floating Calculator: Quickly solve problems with an integrated floating calculator tool.
- Screen Board / Notes: Draw, take notes, and save your work within the app.
- Daily Study Tips: Receive daily expert tips to boost learning and improve study habits.
- Waitlist Management: Capture user name and email for waitlist and beta access; integration with Mailchimp/Formspree.

## Style Guidelines:

- Primary color: Deep navy (#0B0F14) for a modern and sophisticated feel.
- Background color: A slightly lighter shade of navy (#1A202C) to provide contrast.
- Accent color: Neon cyan (#0EA5A0) for interactive elements, providing a futuristic touch.
- Font: 'Inter' sans-serif for a clean and modern aesthetic suitable for both headlines and body text.
- Line icons for features, ensuring a simple and clean interface.
- Mobile-first, responsive design that adapts seamlessly to different screen sizes.
- Subtle animations, such as fading effects on the hero section and sliding fade-ins for features, to enhance user engagement.
